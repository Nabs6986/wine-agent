CLI Commands

Wine Agent includes a command-line interface for backup, restore, and other operations:

wine-agent run Start the web server
wine-agent backup Create a timestamped database backup
wine-agent backup --include-exports Backup with JSON/CSV exports
wine-agent restore <backup-file> Restore database from backup
wine-agent check-config Display configuration status
wine-agent version Show version number
